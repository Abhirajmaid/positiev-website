# Strapi Backend

This folder is for the Strapi backend.

## Setup

1. Create a new Strapi project here or clone an existing one.
2. Configure content types for:
   - Hero (headline, subtext, CTAs)
   - Leadership (members with name, role, image)
   - Investor metrics
   - Partner logos
   - Forms (Franchise, Fleet, OEM, Investors)

## API URL

Set `NEXT_PUBLIC_API_URL` in the client `.env.local` to your Strapi API URL (e.g. `http://localhost:1337/api`).
